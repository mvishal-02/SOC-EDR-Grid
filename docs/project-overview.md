# Project Overview

The Sentient Shield SOC-EDR Grid is a simulated Security Operations Center environment designed to demonstrate enterprise security monitoring capabilities.

The system uses the Wazuh SIEM platform to monitor endpoints, detect suspicious activities, and visualize attack behavior using the MITRE ATT&CK framework.

The project includes:

- SOC infrastructure deployment
- Endpoint monitoring
- File Integrity Monitoring
- Active response automation
- Threat simulation using Atomic Red Team

The environment is connected using Tailscale VPN to provide secure communication between all systems in the lab.
